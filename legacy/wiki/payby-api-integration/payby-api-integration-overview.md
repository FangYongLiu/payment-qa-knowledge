# PayBy 接口文档总览

本页说明 PayBy 接口文档的整体定位、阅读对象与适用范围，便于不同角色快速判断是否需要阅读及如何使用本套文档。

## 文档定位

- 面向商户系统集成 PayBy 的技术性接口文档。
- 覆盖商户接入 PayBy 过程中涉及的接口规范与集成说明。

## 阅读对象

本文档面向商户系统在集成 PayBy 时涉及的以下角色：

- 技术架构师
- 研发工程师
- 测试工程师
- 系统运维工程师

## 适用的商户系统类型

文档适用于以下类型的商户系统接入场景：

- 在线购物平台
- 人工收银系统
- 自动化智能收银系统
- 其他需要接入 PayBy 的商户系统
