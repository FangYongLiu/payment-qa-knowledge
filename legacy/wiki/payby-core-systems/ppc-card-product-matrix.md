# PPC卡产品矩阵

> 状态：骨架（待补充）。本页用一张表汇总 PPC 下所有卡产品的差异字段，避免测试时把不同卡产品的规则混用。

## 用途

- 集中呈现 PPC 下各卡产品的关键区分字段。
- 供 QA 在用例设计、回归、问题定位时快速比对，避免规则串用。

## 覆盖的卡产品

- Salary
- Multi-Currency
- Prepaid

## 建议字段（矩阵列）

- 卡产品名称（Salary / Multi-Currency / Prepaid）
- 卡组织（Mastercard / Jaywan）
- 物理卡 / 虚拟卡
- 支持币种
- 默认限额（单笔 / 日 / 月 / 年）
- KYC 等级要求
- 发卡条件（是否需要劳工合同、护照、Emirates ID 等）
- 目标用户群
- 适用消费场景（ATM / POS / 线上 / 跨境）
- PRD / 文档链接

## 数据来源

- PPC 各卡产品 PRD
- 业务运营对外材料
