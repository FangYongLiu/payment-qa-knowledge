# Smart Code收款

Smart Code 让商户在门店摆放一张二维码即可收款，无需任何硬件；客户扫码支付后，商户和指定员工可即时收到通知。

## 商户旅程

1. 商户将 PayBy Smart Code 摆放在门店显眼位置
2. 顾客扫描 PayBy Smart Code 并完成支付
3. 商户收到支付通知

## 新增 Smart Code

操作路径：[Products] → [Smart Code] → [Add New Code]

新增完成后即可在列表中看到对应的 Smart Code。

## 下载与打印

- 在 Smart Code 详情页，可下载/打印 Smart Code
- 打印后请张贴并使用本商户专属的 Smart Code 收款
- 注意：示例图中的二维码仅用于演示，请勿使用

## 添加 Follower 接收支付通知

商户员工可作为 Follower 接收该 Smart Code 的支付到账通知。

操作路径：

1. 进入 [Products] → [Smart Code] → [Manage]
2. 点击 [+ ADD FOLLOWER]
3. 选择需要 follow 的员工（Staff）
4. 点击 [ENTER] 确认

设置完成后，被选中的员工即可在每笔成功收款时收到支付通知。
