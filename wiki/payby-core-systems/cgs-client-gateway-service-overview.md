# CGS 客户端网关服务总览

CGS（Client Gateway Service）是面向客户端的网关服务，归属于 payby-core-systems 业务域。

## 定位

- 全称：Client Gateway Service
- 简称：CGS
- 类型：客户端网关服务

## 作用

作为客户端请求的统一入口网关，承担来自客户端的接入与路由职能。
