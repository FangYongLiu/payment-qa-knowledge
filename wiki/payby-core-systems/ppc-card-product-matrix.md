# PPC卡产品矩阵

> 一张表说清 PPC 下所有卡产品的差异，避免测试时把不同卡产品的规则混用。

/ 状态：骨架（待补充） / 维护人：Jianfei Wang / 创建于 2026-05-04 /

## 页面用途

汇总 PPC 下各卡产品在发卡、限额、KYC、适用场景等维度的差异，作为测试与业务对照基线，避免不同卡产品的规则被混用。

## 覆盖卡产品

- Salary
- Multi-Currency
- Prepaid

## 对照维度

建议按以下字段建立矩阵：

- **卡产品名称**：Salary / Multi-Currency / Prepaid
- **卡组织**：Mastercard / Jaywan
- **形态**：物理卡 / 虚拟卡
- **支持币种**
- **默认限额**：单笔 / 日 / 月 / 年
- **KYC 等级要求**
- **发卡条件**：是否需要劳工合同、护照、Emirates ID 等
- **目标用户群**
- **适用消费场景**：ATM / POS / 线上 / 跨境
- **PRD / 文档链接**

## 数据来源

- PPC 各卡产品 PRD
- 业务运营对外材料
