# PayBy系统组件总览

PayBy 系统在架构上划分为四大模块：Payment Product、Payment Core、Payment Public 和 Infrastructure，分别承担产品层、核心支付能力、公共服务与基础设施职责。

## 模块划分

- **Payment Product**：支付产品层
- **Payment Core**：支付核心层
- **Payment Public**：支付公共服务层
- **Infrastructure**：基础设施层
