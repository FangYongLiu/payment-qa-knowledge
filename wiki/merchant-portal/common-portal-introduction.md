# 常用Portal介绍

本页介绍运营/管理后台 Portal 的常用功能模块布局，以及「用户管理」界面的常见操作元素。

## 左侧功能模块导航

Portal 左侧导航栏按业务域分组，常见模块包括：

- VIP管理
- 缓存管理
- 用户解绑
- 短信管理
- 收银台鉴权
- 协议配置
- 余额宝
- 产品管理
- BASIS审核
- 综合查询
- 业务管理
- 风控管理
- 营销管理
- 客服系统
- 活动管理
- 系统管理

## 系统管理子模块

系统管理下包含以下子菜单：

- 用户管理
- 角色管理
- 部门管理
- 字典管理
- 菜单管理
- 登录日志
- 操作日志

## 顶部多标签页

Portal 支持多标签页并行操作，常见已开标签示例：

- 加解密查询
- 会员管理
- 用户管理

## 用户管理界面

用户管理页面的主要构成：

### 部门树
左侧以树形结构展示部门层级（如 `测试 / 顶级`），用于按部门过滤用户。

### 搜索区
- 关键字输入框：支持按 `账号/姓名/手机号` 检索
- `注册时间` 筛选输入
- `搜索` 按钮

### 用户数据表
表格字段：

| 字段 | 说明 |
|------|------|
| 账号 | 登录账号 |
| 姓名 | 用户中文名 |
| 性别 | 男 / 女 |
| 邮箱 | 联系邮箱（可排序） |

支持复选框多选，便于批量操作。

### 新增用户
通过「Add User」弹窗进行新用户录入。
