# 出租车POS更换指南

在商户控台通过"Apply For New Device"申请一台 Customized POS，用于替换出租车POS。

## 操作入口

- 登录 PayBy 商户控台
- 左侧导航进入设备申请页面：`Apply For New Device`

## 操作步骤

1. **Store Information**：在 "Add or select a store" 下拉中选择目标门店（如 `ADCB Store`），确认 `Store Name` 与 `Store Location`。
2. **Device Information**：在设备类型卡片中选择 **Customized POS**。
3. 填写 `* Quantity`（默认 `1`）。
4. 点击 **SUBMIT** 提交申请。

## 设备类型说明

申请页提供以下设备选项，更换出租车POS时固定选择 `Customized POS`：

- Smart box
- Smart POS
- Virtual POS
- **Customized POS**（替换出租车POS时使用）
- Face Pay Terminal
