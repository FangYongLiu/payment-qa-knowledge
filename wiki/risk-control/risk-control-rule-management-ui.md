# 风控业务规则管理界面说明

风控管理后台「业务规则」列表页用于检索、维护与排序某一业务域（如 domain=risk-control）下的执行规则，支持按策略、类型、状态、规则ID、描述、变量、标签等多维筛选。

## 顶部筛选区

支持以下筛选条件组合查询（点击「搜索」执行）：

- **所属策略**：文本输入
- **类型**：下拉，默认「不限」
- **启用状态**：下拉，默认「不限」
- **规则ID**：文本输入
- **描述**：文本输入
- **变量名称**：文本输入

## 标签筛选

筛选区下方提供「标签:」快捷筛选，可点击的彩色标签包括：

- `SC`
- `SCC`
- `IP类型`
- `MUP`
- `ATO`
- `AP`

## 操作栏

- **新建业务规则 +**：新增一条业务规则
- **更新排序**：批量保存优先级调整结果
- **每页显示 N 条记录**：分页大小选择（默认 25）
- **Search:**：表格内即时检索框

## 规则列表字段

表格列（均支持升降序排序）：

| 字段 | 说明 |
|---|---|
| 选中全部 | 行级复选框，支持批量操作 |
| 所属策略 | 规则归属的策略编号（如 `CT0005`） |
| 规则ID | 规则唯一标识（如 `VR1110001`） |
| 规则分类 | 例如「执行规则」 |
| 是否短路规则 | 是 / 否 |
| 规则描述 | 文本描述（如「1110 - 支付事件反欺诈」） |
| 优先级 | 数值输入框，配合「更新排序」批量保存 |
| 启用状态 | `已启用` / `已禁用`（禁用为红色标识） |
| 标签 | 命中的标签集合（如 `SCC`） |
| 分数 | 命中规则后赋予的分值 |
| 最后更新人 | 最近一次修改的操作账号 |
| 更新时间 | 最近修改时间戳 |
| 创建时间 | 规则创建时间戳 |
| 操作 | 行级操作入口 |

## 行级操作

每条规则在「操作」列提供以下入口：

- **删除**：移除该规则
- **复制**：基于该规则克隆一条新规则
- **详情**：查看 / 编辑规则明细
