# 风控限额限次流量统计规则配置

本页说明风控后台中"限额限次-流量统计规则"列表的字段含义与示例配置，用于按维度统计实时流量以支撑限额限次风控判断。

## 列表字段说明

| 字段 | 说明 |
| --- | --- |
| 规则编号 | 规则唯一标识，如 `F000001`，可点击进入详情 |
| 规则名称 | 规则描述/用途说明 |
| 统计维度 | 参与流量聚合的字段组合 |
| 时间字段 | 流量时间归属字段，如 `支付时间 (PayTime)` |
| 流量类型 | 当前均为 `实时流量` |
| 接入点 | 形如 `CP-BP-000001-x` 的虚拟接入点编码及名称 |
| 状态 | `ENABLED-已启用`（绿色）/ `DISABLED-已禁用`（红色加粗） |
| 操作 | `查看详情` ｜ `流量数据` |

## 状态枚举

- `ENABLED-已启用`
- `DISABLED-已禁用`

## 时间字段

- 支付时间 (`PayTime`)

## 流量类型

- 实时流量

## 接入点示例

- `CP-BP-000001` - 限额限次流量统计-虚拟接入点
- `CP-BP-000001-1` - 限额限次流量统计-虚拟接入点
- `CP-BP-000001-2` - 限额限次流量统计-虚拟接入点

## 规则示例

### F000001 限额限次-流量统计

- 统计维度：`会员ID`、`资金方向`、`KYC方式`、`币种`
- 时间字段：支付时间 (PayTime)
- 流量类型：实时流量
- 接入点：`CP-BP-000001-1` - 限额限次流量统计-虚拟接入点
- 状态：`DISABLED-已禁用`

### F000013 限额限次：商户端转账到卡流量统计

- 统计维度：`业务产品码`、`会员ID`、`资金方向`、`币种`
- 时间字段：支付时间 (PayTime)
- 流量类型：实时流量
- 接入点：`CP-BP-000001-2` - 限额限次流量统计-虚拟接入点
- 状态：`ENABLED-已启用`

### F000012 限额限次（新）-带业务产品码

- 统计维度：`会员EID`、`业务产品码`、`资金方向`、`币种`
- 时间字段：支付时间 (PayTime)
- 流量类型：实时流量
- 接入点：`CP-BP-000001`
- 状态：`ENABLED-已启用`

## 配置要点

- 规则维度组合不同 → 对应不同的限额限次统计粒度（如是否引入 `业务产品码`、`KYC方式`、使用 `会员ID` 或 `会员EID`）。
- 通过 `状态` 字段控制规则是否生效，禁用后不再参与实时流量统计。
- 同一类业务可通过区分 `接入点`（如 `-1`/`-2` 后缀）拆分独立的统计通道。
