# 金额边界值集

金额相关用例的"边界值卡片"集合，供 LLM 生成测试用例时直接引用，避免每次重新设计边界值。

## 本页用途

- 沉淀金额测试中可复用的边界值
- 作为用例生成的标准输入，保证不同用例对同类边界的取值一致

## 组织方式

按 **用例类型 × 币种** 组织，每条边界值包含以下字段：

- 测试值
- 预期行为（通过 / 拒绝 / 风控）
- 适用场景
- 来源依据（限额 / 监管 / 卡组织）

## 必含的边界值类别

### 限额边界

每个限额维度均覆盖三类取值：

- `=` 限额值
- `< 1 单位`
- `> 1 单位`

### 精度边界

- `0.01`
- `0.001`
- `0.005`（含舍入）
- `9999999999.99`

### 币种相关

- AED 两位精度
- 三位精度币种（如 BHD、KWD）

### 特殊值

- `0`
- 负数
- 超大值
- 超长小数位
- 科学计数法

### 汇率边界

- 极小汇率
- 极大汇率
- 汇率为 `1` 的同币种转换

## 数据来源

- 限额体系页（二层）
- 历史精度类 Bug
- 卡组织规则中的金额规范
