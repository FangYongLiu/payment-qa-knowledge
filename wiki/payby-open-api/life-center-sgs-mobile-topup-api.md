# Life Center SGS 移动话费充值API流程

本页概述 Life Center 在 SGS 渠道下，移动话费充值相关 API 的查询与下单流程，归属业务域 `payby-open-api`。

## 流程范围

涵盖以下四个核心接口流程：

- Query prepaid international：预付费国际号码查询
- Query prepaid mobile Top-up：预付费移动话费充值查询
- Query postpaid mobile bill：后付费移动账单查询
- Place mobile order - Prepaid：预付费移动话费下单

## Query prepaid international

用于查询预付费国际号码的可充值信息（流程图详见原文档配图）。

## Query prepaid mobile Top-up

用于查询预付费移动话费的充值套餐/额度信息（流程图详见原文档配图）。

## Query postpaid mobile bill

用于查询后付费移动号码的待缴账单信息（流程图详见原文档配图）。

## Place mobile order - Prepaid

用于创建预付费移动话费充值订单（流程图详见原文档配图）。

> 注：以上各流程的时序细节以原文档中的流程图为准。
